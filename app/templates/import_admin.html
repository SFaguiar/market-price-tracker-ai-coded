Guia de Formatação
Para Produtos:
  • tipo (string) - ex: Arroz
  • subtipo (string, opcional)
  • marca (string)
  • categoria (string)
  • conteudo_embalagem (número)
  • unidade_medida (kg, g, l, ml, un)

Dica: O sistema aceita chaves antigas ("conteudo_da_embalagem", "und") e normaliza automaticamente.

Para Mercados:
  • nome (string)
  • tipo (string) - Atacado, Supermercado, etc.
  • endereco (string, opcional)
  • cidade (string, opcional)
  • estado (string, opcional) - UF
  • latitude (número, opcional)
  • longitude (número, opcional)
Para Compras (NFC-e):
  • Pode enviar uma lista [ ... ] ou um único objeto { ... }
  • A chave mercado com dados do estabelecimento
  • nfe e data_emissao
  • Lista de itens, cada um contendo a chave produto, quantidade e valor_unitario.
